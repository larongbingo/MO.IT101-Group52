## Project Structure
- `/amper-swing-plugin` - Amper plugin for Swing Form UI, generates the UI code from form files
- `/amper-swing-runtime` - Contains a helper class to initialize the UI


### Note on Swing Form UI
- The .form files are only supported by IntelliJ IDEA's Form Designer. Please use IntelliJ IDEA to edit the form files.
